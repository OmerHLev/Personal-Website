Wordle Automation Demo

This project I made during an online course I did about Selenium in Python.
to challenge myself, I decided to write a functional demo based on what I learned during my course.

In this project, I wrote a pytest framework to run automated scripts on the NYT famous game Wordle.
Within this framework I wrote a few testing scripts, but the major ones are a script that grabs the correct word of the day
and a script that guesses 6 random words, weighted for how common they are, and compares the guess results to the correct word.
This basic demo can be used to ensure that Wordle gives correct results to guesses.

During this project I learned Selenium, Github, Jenkins, Pytest, and other tools I've used for the project.
